# Resolvendo Códigos em Python no Codespace
### Olá!! Aqui são alguns exercícios básicos realizados no próprio Codespace.

Com acesso ao Github Copilot.

# 1 - Verificando Números Pares e Ímpares

Descrição: Como entrada receba um número inteiro e verifique se ele é par ou ímpar.

O que foi estudado:

- Utilização de condicionais em Python (if,else) para realizar verificações.
- Conceito do operador de módulo (%) para verificar se um número é par ou ímpar.
- Exploração do uso de uma ferramenta de IA, como o GitHub Copilot, para otimizar a estrutura do código.

# 2 - Calculando Média de Notas

Descrição: Agora vamos calcular a média de três notas fornecidas na entrada do usuário.

O que foi estudado?

- Uso de variáveis para armazenar dados fornecidos pelo usuário.
- Aplicação de operadores aritméticos.
- Prática na solicitação e manipulação de entrada do usuário.

# 3 - Verificando Palíndromos

Descrição: Vamos testar se uma palavra é palíndromo?

O que foi estudado?

- Manipulação de strings em Python, especialmente invertendo uma string.
- Compreensão de como comparar a string original com sua versão invertida para determinar se é um palíndromo.
- Introdução ao conceito de palíndromos e sua aplicação em problemas de programação.
